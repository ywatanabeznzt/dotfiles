Dot Files
=========

## Installation
``` sh
# for Mac or Linux
git clone https://github.com/ywatanabeznzt/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
sh setup.sh
```

``` bat
rem for Windows
git clone https://github.com/ywatanabeznzt/dotfiles.git %HOMEPATH%\.dotfiles
cd %HOMEPATH%\.dotfiles
setup.dat
```
